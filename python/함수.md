### 정렬

##### sort()와 sorted()의 차

sort()
```python
list.sort()
```
원본을 정렬하고 반환, 리스형 함수이기때문에 리스트만 가능


sorted()
```python
list2 = sorted(list1)
```
- 원본을 정렬하지 않음.
- 정렬 후 새 리스트를 만듬
- 내장 함수이기 때문에 리스트외에도 가능
